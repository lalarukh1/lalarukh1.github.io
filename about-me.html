Hello!

I'm LalaRukh. I designed this site in order to get started with Bulma. I have started to like Bulma, but not so much that I could use it as my go-to css framework. I am still in love with tailwind.

Tailwind has nothing that Bulma or even bootstrap has. Even then, it is the best because you don't have to write your own css classes only to make a small change in color or border or any other pre-defined style from components.

It is true that bulma has got many fresh, trendy out of the box components ready to use, and I have tried using some in this site to see how they can fit different scenarios. But still, I had to write a lot of css to make those components fit in my theme.

The site is not mobile optimized yet but I am working on it!
