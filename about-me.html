Hello!

I'm LalaRukh. I designed this site in order to get started with Bulma. I like it, but not so much that I would start using it as my go-to css framework. I am still in love with tailwind.

Tailwind has nothing that Bulma or even bootstrap has. Even then, it is the best because you don't have to write your own css classes only to make a small change in color or border or any other pre-defined style from components.

It is true that bulma has got many fresh, trendy out of the box components ready to use, and I have tried using some in this site to see how they can fit different scenarios. But still, I had to write a lot of css to make those components fit in my theme. That is really a shame.
